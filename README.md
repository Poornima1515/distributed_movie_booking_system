
# Distributed Movie Ticket Booking System

## Features
- Real-time seat booking
- Redis distributed locking
- Socket.IO live updates
- React frontend
- Node.js backend

## Tech Stack
- React.js
- Node.js + Express
- Redis
- Socket.IO

## Run Backend
```bash
cd backend
npm install
node server.js
```

## Run Frontend
```bash
cd frontend
npm install
npm start
```

## Redis
Make sure Redis server is running:
```bash
redis-server
```
